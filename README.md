# apiNG-plugin-flickr
Flickr Plugin for [apiNG](https://github.com/JohnnyTheTank/apiNG)

## Promise Library
used promise library: [angular-flickr-api-factory](https://github.com/JohnnyTheTank/angular-flickr-api-factory)

## supported apiNG models
- **`social`**
- **`image`**
